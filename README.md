# 👟 Stylish Shoes - E-commerce Website

A modern and responsive e-commerce website designed to showcase stylish footwear with a clean and user-friendly interface.

---


## 📌 Features

* 🛍️ Product showcase UI
* 🛒 Add to Cart functionality (localStorage)
* 🤖 Simple chatbot support
* 📱 Fully responsive design
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
